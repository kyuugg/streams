A clean Twitch multistream viewer built with vanilla HTML, CSS and JavaScript. Save your favorite streamers locally, switch between them with a single click, and watch directly through the embedded Twitch player.

> 🌐 **[Try it live → streams.kyuu.gg](https://streams.kyuu.gg)**

## ✨ Features

- Add and save your favorite Twitch streamers
- Persistent streamer list using `localStorage`
- One-click switching between saved channels
- Embedded Twitch player
- Remove streamers individually
- Clean dark UI inspired by Twitch

## 🛠️ Built With

- HTML
- CSS
- JavaScript (Vanilla)

## 🚀 Getting Started

No build tools or dependencies required.

```bash
git clone https://github.com/kyuugg/streams.kyuu.gg.git
cd streams.kyuu.gg
open index.html
```

## 📌 Notes

This project is ayukyuu's first "real" JavaScript application and one of the first websites I built completely from scratch without following a tutorial. It was created as part of my frontend learning journey and focuses on writing clean, understandable vanilla JavaScript instead of relying on frameworks.

As of **July 2026**, the project includes local storage, dynamic DOM manipulation, event delegation, and Twitch embedding. Future updates may include live status indicators, profile pictures, and additional quality-of-life improvements.
